# AKARI（灯）

大切な人を、いつも心に。

宗教を問わず、亡くなった家族・友人・ペットを静かに偲べるメモリアルアプリです。

## 使い方

`index.html` をブラウザで開くだけで動作します。ビルドや依存パッケージは不要です。

## 主な機能

- メモリアルページ（写真・名前・続柄・誕生日・命日・思い出メモ）
- 命日通知（あと何日／当日表示）
- お供え（花・ろうそく・お線香・果物・お茶・ご飯）
- 手を合わせるモード
- 家族と共有（画面のみ）
- テーマ曲（Apple Music／Spotify／Amazon Musicの検索リンク）
- テキスト／Excel貼り付けによるまとめてインポート

## データ保存

ブラウザの localStorage にのみ保存されます。サーバーへの送信は行いません。
